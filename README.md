# 🚀 Portal Backend API

This is the backend of the Portal project built using Node.js, Express.js and MongoDB.

## 📌 Features
- User Authentication (Login / Register)
- JWT Based Authorization
- Company & Job Management APIs
- Protected Routes
- MongoDB Database Integration

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- dotenv

## 📂 Folder Structure
```
backend/
│── controllers/
│── models/
│── routes/
│── middleware/
│── config/
│── .env
│── server.js
```

## ⚙️ Installation & Setup

1. Clone the repository
```
git clone https://github.com/Meghnaparashar810/codsoft-task-.git
```

2. Install dependencies
```
npm install
```

3. Create a `.env` file and add:
```
PORT=8000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

4. Start the server
```
npm run dev
```

Server will run on:
```
http://localhost:8000
```
